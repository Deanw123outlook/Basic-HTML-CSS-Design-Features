FRONT-END TRACKING SYSTEM (GEOLOCATION - MAPPING)

Client Metadata Tracking Feature

Tool Information
This is a Geo-Location Tool which will use a piece of software code (Pixel Tracker) built using Javascript that has been set up & installed by the client. This tool will connect to our backend servers to update the database. Primarily, we can track our client geo-location aiming to improve use of service.
  • Enter Client Name
  • Enter Client Company Name
  • Enter Client Email-Address
  • Enter Client TELEPHONE
  • Enter Client Unique ID
  • Enter Clent Company
  • Enter Client EIRCODE
  • Click SUBMIT
Next, our tool will map the data to the backend database using Javascript & Python and finally bring us to the GEOLOCATION of our client address.
